Components

This page demos a set of web components created with Lit. The rendered components are followed by their corresponding source code.

Lit is a small library that makes building web components easier - providing support for reactive properties, simpler templating syntax, and more. These components were used to vet the technology, to uncover benefits and drawbacks, and are not proposed as-is for production.

The Internet Archive UX team has adopted Lit and is introducing web components to the site.

Progressive Enhancement

These components support progressive enhancement. You can write standard HTML that works without JavaScript, then automatically enhance it with web components when the script loads.

Form Integration Example

Progressive enhancement shines in forms where native HTML is crucial for accessibility and functionality:

Sign Up

<form>
  <h2 data-ol-heading data-size="title-3">Sign Up</h2>

  <label for="username">Username</label>
  <input type="text" id="username" required>

  <label for="email">Email</label>
  <input type="email" id="email" required>

  <div data-ol-button-group>
    <button type="submit" data-ol-button data-variant="primary">
      Create Account
    </button>
    <button type="button" data-ol-button data-variant="secondary">
      Cancel
    </button>
  </div>
</form>
